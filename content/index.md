### 1. The $4.5M Insider Threat | Crypto & OSINT
**The Challenge:** Identify an internal actor responsible for a multi-year $4.5M fund drain across BTC, ETH, and TRON.
*   **Key Investigative Pivot:** Correlated WebMoney exchanger leaks with internal employee data to unmask the **Head of Anti-Fraud QA**.
*   **Investigation Outcome:** $1.5M in VASP deposits confirmed; full evidence package delivered for prosecution.

### 2. Phobos Ransomware | SQL-to-Blockchain Attribution
**The Challenge:** Attribute physical identities to ransomware wallets using a decrypted 3rd-party exchange database.
*   **Key Discovery:** Identified "address reuse" in internal exchange accounting, distinguishing transit wallets from actor-controlled funds.
*   **Investigation Outcome:** De-anonymized key affiliates using SQL forensics and historical WebArchive logs.
👉 [**Read Deep Dive**](phobos_ransomware.md)

### 3. Darknet Marketplace Forensics | 1TB Server Image
**The Challenge:** Analyze a 975GB server image (`marketplace.com`) with custom-encrypted SQL databases.
*   **The Approach:** Reversed PHP encryption logic to bypass failed automated tools and bulk-decrypted tables using Python.
*   **Investigation Outcome:** Provided Law Enforcement with high-value targets and crypto-wallets not indexed by major CTI vendors.
👉 [**Read Deep Dive**](shadow_service_attribution.md)

### 4. Adversary Attribution: The "Afanasiy" Case

*   **The Hunt:** Pivoted from a "money-for-money" scam pattern to physical residence data, social media, and vehicle registration.
*   **Investigation Outcome:** Full 130-account attribution map (XMind) proving a decade of illicit activity.
